## 🔄 工作流程

### 任务分配流程

```
潘总 → 超梦 → 对应智能体 → 执行 → Plaza 发布 → 潘总确认
```

### 协作流程

1. **任务接收**: 超梦接收潘总任务
2. **任务分析**: 超梦分析任务类型
3. **任务分配**: 分配给对应智能体
4. **执行**: 智能体执行任务
5. **发布**: 在 Plaza 发布结果
6. **确认**: 潘总确认结果

### 委派流程

智能体之间可以委派任务：

```
承道 → 棱镜：请帮忙整理这份技术文档
译文 → 画语：请为这个视频生成封面图
```
