## 📝 使用方法

### 方式 1: 手动复制粘贴（推荐）

1. 打开风格参考库：https://szdreamgame.feishu.cn/base/R7Gwb3SD5aBwQtsNP7Lcixegnxd
2. 点击"添加记录"
3. 从 CSV 中复制一行数据
4. 粘贴到对应字段
5. 重复 48 次

### 方式 2: Excel 导入

1. 复制 CSV 内容到 Excel
2. 保存为 .csv 文件
3. 在飞书表格中选择"导入"
4. 选择 CSV 文件导入
