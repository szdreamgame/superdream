## 📋 使用说明

### 基本格式
```
{前缀}, {角色/场景描述}, {风格}, {质量词}, {后缀}
```

### 默认生成参数 ⭐

| 参数 | 值 | 说明 |
|------|-----|------|
| **模型** | `nano-banana-2` | 默认使用 Nano Banana 2 |
| **画面比例** | `16:9` | 角色图/场景图统一横版宽屏 |
| **默认画风** | `日式 3D 渲染 2D` | 默认画面风格 |
| **生成批次** | `x2` | 每次生成 2 张供选择 |
| **图片尺寸** | `1K` | 标准质量，快速生成 |

**特殊场景**：
- **角色设定图**：比例保持 `16:9`（横版）✅
- **场景设定图**：比例保持 `16:9`（横版）
- **批量测试**：批次可改为 `x4` 或更多

### 使用步骤
1. 选择合适的风格模板
2. 复制前缀、质量词、后缀
3. 填入飞书表格的对应字段
4. 脚本会自动拼接生成最终提示词

## 🎨 通用质量词库

### 分辨率相关
- `1K 分辨率` - 标准质量
- `2K 分辨率` - 高清质量
- `4K 分辨率` - 超高清质量
- `8K 分辨率` - 极致细节

### 光照相关
- `电影级光照` - 专业电影布光
- `自然光照` - 自然光效果
- `柔和光照` - 柔光效果
- `戏剧性光照` - 强烈对比
- `霓虹灯光` - 赛博朋克风格

### 细节相关
- `超细节` - 丰富细节
- `精致细节` - 精细刻画
- `简洁风格` - 简约设计
- `丰富层次` - 多层次表现

### 材质相关
- `树脂材质` - 潮玩质感
- `金属质感` - 金属表面
- `布艺质感` - 纺织品效果
- `玻璃质感` - 透明材质

## 🔧 飞书表格字段配置

### 必填字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 生成参数 - 风格 | 单选 | 从预设风格选择 |
| 生成参数 - 质量 | 单选 | 标准/高清/超高清 |
| 生成参数 - 提示词前缀 | 长文本 | 从模板复制 |
| 生成参数 - 提示词后缀 | 长文本 | 从模板复制 |

### 可选字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 生成参数 - 自定义质量词 | 长文本 | 额外添加质量词 |
| 最终提示词 | 长文本 | 自动拼接（只读） |

## 💡 使用技巧

1. **先测试再批量**：新风格先测试一张，确认效果后再批量生成
2. **保留成功提示词**：效果好的提示词保存到模板库
3. **持续优化**：根据生成效果调整质量词
4. **风格混合**：可以混合不同风格的元素
