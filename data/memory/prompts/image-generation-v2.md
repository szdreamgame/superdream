## 📋 默认生成参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **模型** | `nano-banana-2` | 默认模型 |
| **画面比例** | `16:9` | 横版宽屏（场景图默认） |
| **生成批次** | `x2` | 每次生成 2 张供选择 |
| **图片尺寸** | `1K` | 标准质量，快速生成 |

**特殊场景**：
- **角色设定图**: 比例 `3:4` (竖版)
- **场景设定图**: 比例 `16:9` (横版)

## 🎯 画面描述固定前缀

### 角色设定图前缀
```
生成带特写和正面背面的人物设定图，白色背景，无文字。
```

**用途**: 确保角色设定图包含多角度视图，便于后续制作

### 场景设定图前缀
```
生成场景背景图，无人环境。
```

**用途**: 确保场景图纯净，便于后期合成角色
