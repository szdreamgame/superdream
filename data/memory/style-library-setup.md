## 🔧 配置步骤

### 步骤 1: 打开风格参考库

访问：https://szdreamgame.feishu.cn/base/R7Gwb3SD5aBwQtsNP7Lcixegnxd

### 步骤 2: 添加 48 条记录

1. 点击"添加记录"
2. 填写"风格名称"（如"日式 3D 渲染 2D"）
3. 选择"风格分类"（2D 风格/3D 风格/定格动画/真人风格）
4. 上传缩略图（从云文档中下载后上传）
5. 填写提示词（从上表中复制）

### 步骤 3: 在主表格中添加关联字段

1. 打开主表格：https://szdreamgame.feishu.cn/base/GHrubiTjnayG4fsWP2IcJIotnfc
2. 点击"+"添加字段
3. 选择"关联"字段类型
4. 选择关联到"图片风格参考库"
5. 命名为"**画面风格**"

### 步骤 4: 测试选择功能

1. 在主表格中点击"画面风格"字段
2. 应该能看到风格参考库中的所有风格
3. 选择任意风格，查看是否显示缩略图
