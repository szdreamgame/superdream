# 译文 Agent 消息处理规则

## 📋 配置说明

### 触发条件
在"漫剧 - 文案评审群" (`oc_95a9882e1aca9546c1930b2d27660a6a`) 中：

1. **发送视频文件** + **@译文**
2. **发送视频 URL** + **@译文**
3. **发送命令**：`/文字 [视频文件/URL]`

### 处理流程

```
接收消息 → 检查是否@译文 → 提取视频 → 上传 OSS → 调用 Coze → 返回结果
```
