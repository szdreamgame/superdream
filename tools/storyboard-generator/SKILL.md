# 分镜脚本生成技能 (Storyboard Generator)

## 触发条件

当用户从 Feishu Bitable 提取剧本数据，或发送故事内容并要求生成分镜脚本时触发。

关键词触发：
- "生成分镜"
- "做分镜"
- "分镜脚本"
- "把这个故事做成分镜"
- 或直接发送故事内容后要求输出分镜

## 核心原则

1. **时长控制**：单个分镜总时长≤15 秒（适配 15-60s 短视频）
2. **对话原样**：所有台词必须原样使用，不添加、不删减、不改写
3. **镜头数量**：每个分镜 2-6 个镜头
4. **输出格式**：严格按参考格式输出
5. **数据提取**：从 Bitable 提取时，优先使用"修改后"字段

## 工作流程

### 1. 输入解析

从 Bitable 或用户输入中提取：
- **主要角色**：优先使用"修改后角色"，否则用"主要角色"
- **场景设定**：优先使用"修改后场景"，否则用"场景设定"
- **完整剧本**：优先使用"修改后剧本"，否则用"完整剧本"

### 2. 思考过程（详细技术分析，先输出）

**思考部分必须完整展示推理过程**，参考原 agent 的长度和详细程度：

```
首先，用户提供了对话内容，格式为角色 + 对话，但没有明确情绪。我需要从对话中推断情绪。

对话是：
- A：台词 → 情绪：XX
- B：台词 → 情绪：XX
...

场景需求没有明确提供，但从对话中推断，场景应该是 XX（如室内聊天室、客厅等）。/从表格字段"场景设定：XX"可以确定，场景是 XX。

用户要求生成分镜脚本，对话内容较多，我需要确保单个分镜总时长在 15 秒内。

估计每个对话的时长：一般对话每秒约 2-3 个词（中文约 4-5 字/秒）。

计算总对话时长：
- A: 台词 → X 字 → 约 X 秒
- B: 台词 → X 字 → 约 X 秒
...
总对话时长约 X 秒，这超过了 15 秒。所以根据规则，我需要分成 X 个分镜，每个分镜在 15 秒内。

分镜 1：覆盖前部分对话，直到约 X 秒。
分镜 2：覆盖剩余对话。

但对话是连续的，所以我需要合理分割。

分镜 1 结束在 XX 的"XX"之后，XX 的回应在分镜 2。这样分镜 1 时长约 X 秒，分镜 2 时长约 X 秒，均在 15 秒范围内。

现在，输出格式：每个分镜列出镜头，每个镜头描述景别、镜头类型、运镜方式、视角和画面描述。

我需要为每个分镜设计 2-6 个镜头。基于对话情绪和内容。

首先，推断角色和场景：
- 角色 A：XX 描述
- 角色 B：XX 描述
- 场景：XX

分镜 1: 对话从 XX 到 XX。镜头设计：
- 第一个镜头：A 说话，XX 情绪。
- 第二个镜头：B 回应，XX 情绪。
...

分镜 2: XX 说最后一句，XX 震惊。镜头设计：
- 第一个镜头：A 说话 XX。
- 第二个镜头：B 震惊。
...

最后，输出时，先分镜 1，然后分镜 2。
```

**思考部分关键要素**：
1. 情绪推断过程（从对话内容推断，而非直接给出）
2. 场景推断过程（从对话或表格字段推断）
3. 逐句时长估算（每句单独计算，标注字数）
4. 分镜分割逻辑（总时长>15s 时如何分割）
5. 镜头设计思路（每个分镜 2-6 个镜头）
6. 画面构建思路（角色动作、场景元素）

### 3. 分镜脚本输出格式

```
分镜 1（X 秒）

镜头 1-1｜景别｜运镜方式｜角色 + 情绪地说："台词"
画面：描述画面内容

镜头 1-2｜景别｜运镜方式｜角色 + 情绪地说："台词"
画面：描述画面内容

---

分镜 2（X 秒）

镜头 2-1｜景别｜运镜方式｜角色 + 情绪地说："台词"
画面：描述画面内容

...
```

**格式要素**：
- 分镜标题：`分镜 X（X 秒）`
- 镜头格式：`镜头 X-X｜景别｜运镜方式｜台词`
- 画面描述单独一行，以`画面：` 开头
- 用 `---` 分隔分镜
- 台词用引号 `"` 包裹

## 镜头设计规范

| 元素 | 可选值 | 说明 |
|------|--------|------|
| 景别 | 远景/全景/中景/近景/特写 | 根据情绪和对话重要性选择 |
| 运镜方式 | 固定/推/拉/摇/移/跟/升降/旋转 | 15s 短视频以固定镜头为主 |
| 视角 | 客观/主观/俯视/仰视 | 默认客观视角 |

## 画面构建规则

1. **角色元素**：从角色描述中提取服装、发型、体型、年龄特征
2. **场景元素**：从场景设定中提取道具、光线、氛围
3. **动作描述**：基于对话情境推断角色动作（递东西、转身、表情变化）
4. **情绪匹配**：台词情绪与画面表情一致（严肃/好奇/淡定等）
