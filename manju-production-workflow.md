## 📝 详细流程

### 阶段 1: 脚本创作

**执行者**: 超梦/承道  
**位置**: 云端

**输入**:
- 原始素材（小说/漫画/剧本）
- 风格要求（搞笑/悬疑/言情等）
- 时长要求（30 秒/60 秒/90 秒）

**输出**:
- 完整脚本（含台词）
- 角色列表
- 场景描述

**工具**:
- Coze 工作流（文本处理）
- 飞书文档（协作编辑）

**完成标准**:
- [ ] 脚本通过审核
- [ ] 角色设定清晰
- [ ] 场景描述完整

---

### 阶段 2: 分镜设计

**执行者**: 棱镜  
**位置**: 云端

**输入**:
- 阶段 1 的脚本

**输出**:
- 分镜表格
- 每镜时长
- 角色/场景标注

**工具**:
- 飞书表格（分镜管理）
- 画语（可选，生成参考图）

**分镜表格字段**:

| 镜号 | 时长 | 场景 | 角色 | 台词 | 备注 |
|------|------|------|------|------|------|
| 001 | 3s | 教室 | 小明 | "早上好！" | 全景 |
| 002 | 2s | 教室 | 小红 | "早！" | 近景 |

**完成标准**:
- [ ] 所有镜头分配完毕
- [ ] 时长合理（总时长符合要求）
- [ ] 角色/场景标注清晰

---

### 阶段 3: 语音识别（可选）

**执行者**: 译文  
**位置**: 云端

**适用场景**:
- 有参考音频需要转写
- 需要提取角色台词

**输入**:
- 音频文件/URL

**输出**:
- 文字稿
- 角色台词分离

**工具**:
- 阿里云 NLS
- Coze 语音识别

**完成标准**:
- [ ] 文字稿准确率 > 95%
- [ ] 角色台词正确分离

---

### 阶段 4: 视频生成 ⭐

**执行者**: 御影  
**位置**: Mac Mini（本地）

**输入**:
- 阶段 2 的分镜表格
- 阶段 1 的脚本

**输出**:
- 成品视频（MP4）
- SeaDance 项目链接

**工具**:
- SeaDance 网页 (https://seedance.ai)
- Chrome 浏览器

**操作步骤**:

1. **登录 SeaDance**
   - 打开 seedance.ai
   - 使用账号登录
   - 确认登录状态

2. **创建新项目**
   - 点击"新建项目"
   - 填写项目名称
   - 选择视频比例（9:16/16:9）

3. **上传脚本/分镜**
   - 导入阶段 1 的脚本
   - 或手动输入台词

4. **配置角色**
   - 选择角色形象
   - 配置服装/发型
   - 设置表情

5. **配置场景**
   - 选择背景场景
   - 添加道具（可选）

6. **生成预览**
   - 点击"预览"
   - 检查效果
   - 调整细节

7. **渲染导出**
   - 点击"渲染"
   - 等待完成
   - 下载 MP4 文件

**完成标准**:
- [ ] 视频渲染完成
- [ ] 画质清晰
- [ ] 台词同步准确
- [ ] 文件已保存到本地

---

### 阶段 5: 成品发布

**执行者**: 御影  
**位置**: Mac Mini（本地）

**输入**:
- 阶段 4 的成品视频

**输出**:
- 飞书群发布的视频
- 飞书表格的完成记录

**工具**:
- 飞书网页/App
- 飞书开放 API

**操作步骤**:

1. **上传视频到飞书**
   - 打开目标群聊
   - 上传视频文件
   - 添加说明文字

2. **更新制作进度表格**
   - 标记状态为"已完成"
   - 填写完成时间
   - 添加视频链接

3. **通知相关人员**
   - @潘总 审核
   - @相关人员 查看

**完成标准**:
- [ ] 视频已发布到群聊
- [ ] 进度表格已更新
- [ ] 相关人员已通知

---

### 阶段 6: 反馈收集

**执行者**: 超梦  
**位置**: 云端

**输入**:
- 群聊反馈
- 审核意见

**输出**:
- 反馈汇总
- 改进建议

**工具**:
- 飞书消息监听
- 飞书表格

**完成标准**:
- [ ] 反馈已记录
- [ ] 改进建议已提出
- [ ] 下次制作已优化

## 🚨 异常处理

### 常见问题及解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| SeaDance 登录失败 | Cookie 过期 | 重新登录，更新 Cookie |
| 视频渲染失败 | 脚本格式错误 | 检查脚本格式，重新提交 |
| 飞书发布失败 | 文件过大 | 压缩视频或分集发布 |
| 语音识别不准 | 音频质量差 | 重新录制或使用更清晰音源 |

### 上报流程

```
发现问题 → 记录到飞书表格 → @超梦协调 → 分配处理 → 解决后更新状态
```
