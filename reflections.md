### 2026-03-20 20:00 - 承道模型问题诊断

**触发器**: 用户报告承道模型异常

**决策过程**:
1. **问题定位**
   - 检查承道配置：显示 foxcode-ultra/claude-opus-4-6
   - 检查实际调用日志：发现使用 foxcode-ultra 但返回空响应
   - 测试 API：发现 pry 提供商返回 "overloaded" 错误

2. **根因分析**
   - pry 提供商 API 不稳定（服务过载）
   - 承道配置与预期不符（应该是 MiniMax-M2.5）

3. **解决方案**
   - 切换回稳定的 MiniMax-M2.5
   - 修正 openclaw.json 配置
   - 重启 Gateway 应用配置

**工具调用**:
```
- exec: ps aux 检查 Gateway 进程
- exec: cat 检查配置文件
- exec: openclaw gateway restart
```

**学习成果**:
- 配置修正后必须重启 Gateway
- 承道的 subagent 可能继承主智能体模型，需要显式配置
